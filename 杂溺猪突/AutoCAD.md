AutoCAD

---

## 一、CAD 基础

1、什么是 CAD

+ CAD 是绘制机械图、建筑图、装修图等二维复杂工程图的不二之选。二维设计软件的王者。
+ CAD 可以绘制平面图、轴测图（二维线框表示三围图形）、立体图（三维图形）。
+ 只要从事机械、建筑、家装、电子等行业，无论是技术人员还是一线工人，都离不开 CAD。

2、CAD 界面及编辑

- 界面分为快速访问工具栏、开始工具、工具栏、绘图区、命令栏、辅助状态栏。
- 工具栏上右键可以对分区进行显示和隐藏设置。选项卡右边小三角，可以让工具栏最小化。
- 鼠标在工具图标上停留，可以弹出相应帮助。
- 点击辅助状态栏里的齿轮图标可以切换工作空间，有恢复所有界面设置的作用。

3、CAD 新建打开及保存

- 新建文件有多种方式，快速访问工具栏里新建、开始里新建、快捷键 **ctrl + N**，一般我们选择*无样板公制*。（在新建旁边的小三角菜单里面）
- 界面最上方显示的是当前文件的文件名，标准 CAD 文件后缀名是 dwg。
- 保存可以点击快速访问工具栏里的保存，或用快捷键 **ctrl + S**.
- 保存可以自定义路径、名称以及版本。

4、鼠标基础操作

+ 鼠标左键是点击选中。
+ 鼠标中键前滚是放大，后滚是缩小，它是以鼠标为中心的。按住不放拖动鼠标是平移，双击中键是显示全部。
+ 鼠标右键是菜单键。

5、CAD 选择技巧

+ 点选：左键点击，shift + 左键点击是取消选择，ESC 是退出。
+ 矩形框选：左键点击对角点，由左往右是框选住全部才能选中对象，由右往左是框选住部分即可选中对象。
+ 全选：快捷键 **CTRL + A**，当执行某个命令，让选择对象的时候，也可以直接输入 **all**，然后空格是全选。
+ 栏选：执行命令时，提示选择对象，输入 F，画折线，所有与折线相交的对象被选中。

6、CAD 直线工具

+ 直线工具创建的对象本质是线段，而不是无限延伸的直线。、
+ 注意命令提示，可以空格或者回车键来确认退出，而 ESC 是直接退出。
+ 快捷键 L：直接在界面里输入 L，养成使用快捷键的习惯。
+ 动态辅助快捷键 F12，tab 键切换，注意角度需要用回车键确认的。
+ 注意辅助状态的图标，回撤操作可以使用快捷键 **CTRL + Z**。 

7、CAD 相对坐标

+ 绝对坐标：指该点相对于原点的 X（水平）/ Y（垂直）的坐标。
+ 相对坐标：指该点相对于上一点的 X（水平）/ Y（垂直）的坐标值，以 `(x,y)` 的形式输入，注意中间的逗号是英文半角符号。X 轴左负右正，Y 轴上正下负。
+ 与动态输入的区别：动态输入是输入长度和角度，用 tab 键切换；相对坐标是输入水平相对距离和垂直相对距离，用英文逗号切换。
+ 极坐标：带有角度的坐标值，以（距离 < 角度）形式输入，角度指的是直线与水平向右方向的夹角。角度值逆时针为正，顺时针为负。
+ 空格可以重复上一命令。
+ 捕捉设置：可在捕捉图标上右键设置捕捉各种类型。

8、二维结构图：圆

+ 快捷键：C
+ 默认是指定圆心半径作圆，命令行括号内的值是当前的默认值。只有进入某个命令的时候才能捕捉圆心。

9、二维结构图：圆弧

+ 快捷键：ARC
+ 圆弧形成的方向默认是逆时针，可以按住 ctrl 键来切换方向。夹角指的是圆弧两个端点与圆心连线所成的夹角。
+ 弧长指的是圆弧两端点之间的距离，最大不能超过圆弧的直径，否则会绘制失败。

10、二维构图：椭圆

+ 快捷键：EL
+ 椭圆的角度是根据起点的位置，逆时针慢慢增加的。

11、二维构图：矩形

+ 快捷键：rec
+ 矩形工具输入方式：(x,y) ，跟相对坐标一样，是分正负的，X 轴左负右正，Y 轴上正下负。

12、二维构图：多边形

+ 快捷键：pol
+ 多边形工具只能绘制正多边形，即边长及内角都相等。多边形的边数必须大于等于 3。

13、二维构图：多线

+ 快捷键：ml
+ 对正：上，在光标下方绘制多线，参照线是上面的线。无，光标在中间绘制多线，参照线即两线的中心线。下：在光标下方绘制多线，参照线即最下面的线。
+ 多线编辑方式在菜单里：修改 --> 对象 --> 多线。

13、二维构图：构造线及射线

+ 构造线快捷键：xl
+ 构造线是无限延伸的直线，一般用作辅助线。构造线要用鼠标从右到左去框选，选择部分即可选中对象。
+ 射线快捷键：ray。

14、二维构图：多段线

+ 快捷键：pl
+ 多段线是一个整体，可以连续绘制直线和圆弧，多段线也可以修改宽度。

15、二维构图：样条曲线

+ 快捷键：spl
+ 一般用于剖物面的分界线。

16、二维构图：图案填充

+ 快捷键：h
+ 创建方式：A：选择封闭区域内部的点。B：顺时针或逆时针选择边界创建，不封闭区域也可以创建，但是会有图案信息丢失。



## 二、CAD 编辑工具

1、移动工具：

+ 快捷键：M
+ 编辑工具一般可以先选择编辑对象，然后输入快捷键激活命令。注意基准点的选取。

2、旋转工具：

+ 快捷键：RO
+ 同样先选择对象，然后选择基点，即对象绕着哪一点进行旋转。
+ 旋转自带可选的复制功能。
+ 旋转参照：指定基点，输入 R 启动参照，再次选择基点，选择对象上任意一点，再点选参照对象上的点。

3、修剪及延伸工具

+ 修剪快捷键：TR
+ 延伸工具：EX
+ 创建方式：输入快捷键，直接按两次空格，即默认选择所有对象，点选修剪或延伸的对象。
+ 修剪时注意：只有交点而没有突出部分的线是无法修剪的，只能手动删除。
+ 延伸注意，视窗能看得到的边界才能延伸成功。

4、删除及复制工具

+ 删除快捷键：E
+ 复制快捷键：CO
+ 删除时先选中删除的对象，然后输入快捷键 E，按空格键确认删除。
+ 复制工具一次可以复制多个，复制基点：选择的与对象相对位置不变的点。
+ 在命令中需要定位的时候可以利用拾取点进行精确定位。
+ 复制工具具有阵列功能，可以复制多个对象，其中阵列里面的布满命令较为常用。

5、镜像工具

+ 快捷键：MI
+ 先选择要镜像的对象，确认后选择镜像上的两点，确认后完成。

6、倒圆角工具

+ 快捷键：F
+ 启动倒圆角工具之后要检查圆角默认值是否需要更改，倒角设置为 0，可以起到建立直角的作用，可以让不连接的直线直接直角连接。

7、分解工具

+ 快捷键：X
+ 先选择对象，然后输入快捷键 X，将整个对象分解为可以单独编辑的对象。一个整体的图形是无法进行更改属性等操作的，要分解后才可以。

8、拉伸工具

+ 快捷键：S
+ 选择拉伸的对象的时候一定要从右往左选择，且选择部分对象，而从左往右选择是移动功能。

9、缩放工具

+ 快捷键：SC
+ 与中键的区别，缩放工具改变了图形真实尺寸
+ 比例因子有多种输入方式，可以输入分数式，新值在前，旧值在后。
+ 自带参照功能，起到对齐的作用。先选择要缩放的参照边，再选择被参照对象。

10、倒角工具

+ 快捷键：cha
+ 倒角距离 1 对应的就是第一次选择的直线，倒角距离 2 对应的是第二次选择的直线。
+ 以角度倒角指的是第一次选择的直线与水平向右所成的夹角，并没有正负之分。

11、矩形阵列

+ 快捷键：AR
+ 创建方式分为矩形阵列、路径阵列和环形阵列。
+ 阵列工具里，任何数值的改变必须用回车键进行确认。
+ 注意关联选项，默认是开启的，阵列出来的图形是一个整体的。
+ 路径阵列不需要对象在路径上，注意路径阵列里分定距等分和定数等分。
+ 环形阵列可以更改旋转方向。环形阵列时，默认对象随之发生旋转。

12、偏移工具

+ 快捷键：O
+ 与定距拾取功能类似，不同的是偏移是复制对象原本特性到指定距离方向处。
+ 偏移对象是整体时会整个轮廓发生偏移。

## 三、CAD 精准绘图

1、正交模式

+ 快捷键：F8
+ 只能捕捉到水平或垂直的方向，是一种特殊的捕捉模式。
+ 可以利用极坐标或者相对坐标画斜线。

2、极轴追踪

+ 快捷键：F10
+ 可以设置多种角度捕捉。一般绘图中是常开状态。

3、对象捕捉

+ 快捷键：F3
+ 帮助我们快速定位到精确的点。对象捕捉图标上右键可以设置捕捉类型，除了最近点，其它捕捉类型一般都是常开状态。

4、对象捕捉追踪

+ 快捷键：F11
+ 对象捕捉就是指绘图时鼠标靠近该类型点会出现提示，比如端点，中点等。
+ 对象追踪是指你把鼠标停在一个点上片刻，出现小十字符号时移开鼠标，每次鼠标经过该点的水平或垂直方向时会出现虚线提示，只是一次有效。

5、等轴侧捕捉

+ 用于绘制轴测图，轴测图指用二维线框表示三维模型。
+ 必须配合正交模式使用。切换各个面快捷键：F5.

## 四、CAD 注释及标注

![注释及标注图示](..\pics\zhishi.png)

1、线性标注

+ 快捷键：DLI
+ 标注由尺寸线、尺寸界限、尺寸文字组成。
+ 标注两点之间的水平距离或者垂直距离，根据鼠标来进行判断。
+ 线性标注里的多行文字可以输入数字或字母。
+ 可以用文字功能对标注文字更改，也可以直接双击标注文字进行编辑。注意，更改后不再显示真实值，应急处理不推荐。
+ 线性标注里的角度和旋转功能不一样，一个针对的是文字，一个针对的是尺寸界限。
+ 尺寸文字不允许有线穿过，可以采用倾斜样式。注释 --> 标注 --> 倾斜

2、对齐标注和角度标注

+ 对齐标注快捷键：dal，标注两点之间的直线距离
+ 角度标注快捷键：dan，标注两斜线之间的夹角，可以标注弧线的角度。

3、半径标注和直径标注

+ 半径标注：dra
+ 直径标注：ddi
+ 直径符号文字输入：%%C。

4、标注样式

+ 标注样式管理器快捷键：D
+ 默认的标注样式不要更改，要新建一个属于自己的标注样式。

5、引线标注

+ 快捷键：LE
+ 单引线标注在工具栏里找不到，只能快捷键启动。
+ 角度符号输入：%%d
+ 引线标注里文字宽度为 0，指的是采用文字的默认宽度。

6、全局比例

+ 自带的标注样式管理器里面的参数大多都不需要更改，都是国家标准。
+ 当文字或者箭头太小，应新建一个标注样式设置全局比例来调整整个标注样式。
+ 全局比例只能用于当前文档。

7、单行及多行文字

+ 单行文字快捷键：DT
+ 单行文字可以设置旋转角度，每一行都是一个单独可以编辑的对象。
+ 多行文字快捷键：T
+ 多行文字创建的文字是一个整体，双击即可进入编辑。完成输入之后可以按 CTRL+ENTER 确认完成。

8、文字样式

+ 中文不支持大字体
+ 跟标注样式类似，当前样式不允许删除，自带的标准文字样式不要更改，新建一个属于自己的文字样式。
+ 当遇到文字显示 ？的时候，可以更改文字样式里的字体。

## 五、CAD 图层编辑

1、特性工具

+ 可以更改对象的颜色、线宽、线型。
+ 没有的线型可以点击：其他 --> 加载。
+ 特性匹配快捷键：MA
+ 当线宽改变后要打开辅助状态栏里的线宽才能看到。

2、新建图层

+ 图层：类似于 PS 里的图层概念，一层一层透明的纸张叠加到一起的显示效果，这样可以把我们的图形按照层的概念进行分类。方便我们的单独进行编辑。

3、图层样板

+ 新建的图层是无法用于新建的文档。我们可以将设置好的图层信息导出为样板文件 DWT。新建文件的时候直接选择我们设置好的样板文件进行新建即可。
+ 把设置好的样板文件保存到 U 盘里，形成一个良好的习惯，不用每次去设置绘图环境。

4、图层的运用

+ 图层的线型标准
+ 粗实线：可见轮廓线
+ 细实线：尺寸线，尺寸界限，剖面线，引出线
+ 波浪线：断裂处的边界线，视图和剖视的分界线
+ 虚线：不可见的轮廓线
+ 点划线：轴线，对称中心线
+ 双点划线：假想投影轮廓线，中断线

5、图层操作

+ 开关冻结隔离锁定图层共有八种操作，是一一对应的关系。
+ 关闭图层主要为了保护该图层里面的对象。
+ 冻结图层无法冻结当前图层
+ 锁定图层之后能画进去对象，但是无法删除或者编辑。
+ 隔离图层的作用是单独拿出一个图层来编辑，其他图层全部锁定。

6、非连续线性比例

+ 快捷键：lts
+ 功能就是改变非连续线性的显示比例，针对的是全局的
+ 在特性工具栏里更改的是针对单独的某一个对象。
+ 可以用 MA 特性匹配工具进行多个对象调整。

## 六、CAD 块

1、块的认识

+ 可以理解块工具是批量复制，选择性粘贴，很多基于插件或者图库都是基于块工具开发的。

2、块的建立

+ 打开一张图档，选定要创建块的对象。
+ 单击“绘图 --> 块 --> 创建”命令，弹出块定义对话框。
+ 在块定义对话框中，输入“名称”，可以自定义，单击“拾取点”，指定块的插入点。
+ 快捷键：B

3、块的插入

+ 插入时候可以更改块的比例及角度
+ 创建永久块：W
+ 新建文件导入永久块需要单个选择，无法批量导入。

4、属性块

+ 简单的说，属性块就是在图块上附加一些文字属性，这些文字属性可以方便地修改值。
+ 先建立可以更改定义的属性，然后将定义的属性和块连接起来。

5、动态块

+ 动态块的参数分为三块：参数、动作和约束。
+ 参数：包括点、线型、极轴、旋转、翻转、对齐、阵列、可见性、查询列表等。查询列表可以设置与图形的一个多个参数关联，对这些参数进行参数化控制。
+ 动作：移动、缩放、拉伸、阵列、翻转、查询等。
+ 约束：就是调用参数化相关功能，可以对块内图形添加垂直、相切、相等的约束条件，或者利用参数化控制图形尺寸。

## 七、CAD 输出与打印

1、导出多种格式

+ CAD 可以导出多种格式，比如 dwg, dxf, dws, dwt。第一个比较常用的标准文件格式。
+ dwf 格式，这种是选择打印机来保存的，CAD 自带的，不过保存之后就不能修改了。
+ 同时可以导出为图片格式、PDF 格式，这些也是不能直接修改的，一般用来打印成图。

2、模型空间打印

+ 显示：打印出视窗显示的图形。
+ 范围：打印画布中所有图形
+ 窗口：选择打印的范围
+ 打印快捷键：CTRL + P。

3、布局打印

+ 可以新建多个布局，设置好之后可以批量打印。
+ 一个布局里可以新建很多视口进行补充表达。
+ 视口和视口之间是相互独立的，一次只能单独编辑一个视口。
+ 双击视口内部即可激活，退出视口双击空白的地方即可。
+ 小视口不好激活的时候可以移动出来，编辑后再放回去。
